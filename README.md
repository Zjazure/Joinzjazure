# 湛江一中IT社网络报名系统 - PHP重制版

## 安装

1. 克隆本Git库或下载源码，将JoinzjazurePHP目录的内容放置于Web目录中。

2. 本程序的数据库配置位于*JoinzjazurePHP/config.json*内，请打开该文件填入正确的数据库配置。

3. 本程序使用MySQL数据库系统。由于本程序会在数据表不存在时创建用户，设置用户时，请额外给该用户添加mysql数据库的SELECT权限。
